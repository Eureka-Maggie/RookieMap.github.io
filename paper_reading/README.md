# 文献阅读

文献阅读是科学研究中至关重要的部分。对于刚入门的同学，文献阅读的时间应该比做实验更多（引自孙飞老师）

本模块的主要目的是向大家分享如何寻找paper、如何阅读paper

请根据侧边栏的导航信息，或下述列表，查看具体模块的信息：

- [Google scholar](./google_scholar)
- [arxiv](./arxiv)
- [会议列表及主页](./conferences)
- [作者主页](./authors)
